# 화면 OCR 사용 가이드

## 설치 방법

### 1단계: 프로그램 다운로드

GitHub Releases 페이지에서 최신 버전의 설치 프로그램을 다운로드하세요:
- 파일명: `화면OCR_설치프로그램_v1.0.0.exe`
- 크기: 약 150MB

### 2단계: 설치 실행

1. 다운로드한 파일을 더블클릭
2. "이 앱이 디바이스를 변경하도록 허용하시겠습니까?" → **예** 클릭
3. 설치 마법사를 따라 진행
4. 설치 옵션:
   - ✅ 바탕화면 아이콘 만들기 (선택 사항)
   - ✅ Windows 시작 시 자동 실행 (선택 사항)
5. **설치** 버튼 클릭

### 3단계: 설치 완료

설치가 완료되면 자동으로 프로그램이 실행됩니다.

## 사용 방법

### 기본 사용법

#### 1. 프로그램 실행
- 바탕화면 아이콘을 더블클릭
- 또는 시작 메뉴에서 "화면 OCR" 검색

프로그램이 실행되면 콘솔 창이 나타나며 다음과 같은 메시지가 표시됩니다:
```
============================================================
🚀 화면 OCR 프로그램 시작!
============================================================
📌 단축키:
   Ctrl + Shift + C  →  영역 선택 후 텍스트 인식
   Ctrl + Shift + Q  →  프로그램 종료
============================================================
✅ Tesseract 경로: C:\Program Files\ScreenOCR\tesseract\tesseract.exe
============================================================

대기 중... (Ctrl+Shift+C를 눌러 시작하세요)
```

#### 2. OCR 시작
`Ctrl + Shift + C` 키를 동시에 누릅니다.

화면이 반투명 검은색으로 바뀌고 상단에 안내 메시지가 표시됩니다:
```
마우스로 드래그하여 영역을 선택하세요 (ESC: 취소)
```

#### 3. 영역 선택
마우스로 텍스트가 있는 영역을 드래그합니다.
- 빨간색 테두리로 선택 영역이 표시됩니다
- 드래그 중에도 크기 조정 가능
- ESC 키로 취소 가능

#### 4. 텍스트 인식
마우스 버튼을 놓으면 자동으로:
1. 화면 캡처
2. 텍스트 인식
3. 클립보드에 복사

콘솔 창에 인식된 텍스트가 표시됩니다:
```
============================================================
✅ 텍스트 인식 완료! 클립보드에 복사되었습니다.
============================================================
여기에 인식된 텍스트가 표시됩니다
============================================================
```

#### 5. 붙여넣기
원하는 곳에서 `Ctrl + V`를 눌러 붙여넣습니다.

## 실전 활용 예시

### 예시 1: PDF 문서 텍스트 추출
1. PDF를 화면에 띄움
2. `Ctrl + Shift + C`
3. 복사하고 싶은 텍스트 영역 드래그
4. Word, 메모장 등에 `Ctrl + V`로 붙여넣기

### 예시 2: 이미지 속 텍스트 추출
1. 이미지 파일 열기 (JPG, PNG 등)
2. `Ctrl + Shift + C`
3. 텍스트 영역 선택
4. 바로 편집 가능한 텍스트로 변환

### 예시 3: 동영상 자막 추출
1. 동영상 일시정지
2. `Ctrl + Shift + C`
3. 자막 영역 선택
4. 텍스트로 저장

### 예시 4: 웹페이지 복사 금지 우회
1. 복사가 안 되는 웹페이지 열기
2. `Ctrl + Shift + C`
3. 원하는 텍스트 영역 선택
4. 클립보드에서 사용

## 팁과 요령

### 인식률 높이는 방법

1. **선명한 텍스트 선택**
   - 가능한 한 큰 글씨 선택
   - 흐릿한 이미지는 확대 후 캡처

2. **배경 단순화**
   - 텍스트와 배경의 대비가 높을수록 좋음
   - 복잡한 배경은 인식률 저하

3. **정확한 영역 선택**
   - 텍스트만 포함되도록 선택
   - 여백을 최소화

4. **한 번에 적당량만**
   - 너무 많은 텍스트는 나눠서 인식
   - 한 번에 10-20줄 정도가 적당

### 단축키 활용

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `Ctrl + Shift + C` | OCR 시작 | 어디서든 바로 실행 |
| `ESC` | 선택 취소 | 영역 선택 중 취소 |
| `Ctrl + Shift + Q` | 프로그램 종료 | 완전 종료 |

## 문제 해결

### 문제: 텍스트가 인식되지 않아요

**해결 방법:**
1. 텍스트가 충분히 크고 선명한지 확인
2. 영역을 더 크게 선택해보세요
3. 화면 해상도를 높여보세요
4. 텍스트와 배경의 대비를 확인하세요

### 문제: 한글이 깨져서 나와요

**해결 방법:**
1. 프로그램을 완전히 종료 후 재시작
2. 한글 폰트가 명확한 영역을 선택
3. 손글씨는 인식률이 낮을 수 있습니다

### 문제: 단축키가 작동하지 않아요

**해결 방법:**
1. 프로그램이 실행 중인지 확인 (콘솔 창 확인)
2. 관리자 권한으로 실행해보세요:
   - 프로그램 아이콘 우클릭
   - "관리자 권한으로 실행" 선택
3. 다른 프로그램과 단축키가 겹치는지 확인

### 문제: 프로그램이 느려요

**해결 방법:**
1. 선택 영역을 작게 해보세요
2. 다른 프로그램을 종료하고 시도
3. 컴퓨터를 재시작해보세요

### 문제: 설치가 안 돼요

**해결 방법:**
1. 관리자 권한으로 설치 프로그램 실행
2. 백신 프로그램이 차단하는지 확인
3. 디스크 공간이 충분한지 확인 (최소 200MB)

## 제거 방법

1. **제어판에서 제거**

   - 시작 메뉴 → "화면 OCR" → "화면 OCR 제거" 클릭

## 자주 묻는 질문 (FAQ)

**Q: 오프라인에서도 사용할 수 있나요?**
A: 네! 인터넷 연결 없이 사용 가능합니다.

**Q: 다른 언어도 인식되나요?**
A: 현재는 한글과 영어만 지원합니다.

**Q: 손글씨도 인식되나요?**
A: 인쇄된 텍스트에 최적화되어 있습니다. 손글씨는 인식률이 낮습니다.

**Q: 여러 대의 컴퓨터에 설치할 수 있나요?**
A: 네! 학교나 회사에서 자유롭게 사용하세요.

**Q: 보안은 안전한가요?**
A: 모든 처리가 로컬 PC에서 이루어지며, 인터넷으로 데이터가 전송되지 않습니다.

**Q: 업데이트는 어떻게 하나요?**
A: GitHub Releases 페이지에서 새 버전을 다운로드하여 재설치하세요.
